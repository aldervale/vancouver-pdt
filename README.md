# Vancouver PDT

A data visualization showing how permanent daylight time (UTC−7) reshapes daylight across the year for Vancouver residents.

B.C. adopted permanent PDT after the final spring-forward on March 8, 2026.

## Dev

```bash
npm install
npm run dev
```

## Deploy

```bash
fly deploy
```

App is configured for Fly.io (yyz region). Update `fly.toml` with your final app name before first deploy.
